Les balises

Elles permettent au navigateur de comprendre la structure de la page. L'utilisateur, lui, ne verra que le contenu compris dans ces balises.

Les balises ouvrantes & fermantes :

<div> (balise ouvrante) Contenu <div> (balise fermante) Les balises autofermantes : <img>(balise autofermante) Les attributs de balises Les attributs permettent de préciser les propriétés des éléments HTML. Pour le cas de <img>, l'attribut « src » permet de préciser la source (provenance) de l'image ou le « alt ». Si ce dernier attribut est obligatoire, il en existe des facultatifs comme l'attribut « title » permettant de renseigner sur la nature ou le contenu de l'image avec une infobulle. Ces différents attributs peuvent être additionnés et sont séparés par des espaces. Ils fonctionnent sur un système de clé/valeur. <img src="fichier.jpg" alt="Description de l'image" title="Mon image">

Les sélecteurs id et class

Les sélecteurs permettent d'interagir avec le contenu avec CSS et JS.

L'attribut « id » identifie un élément de manière unique, alors que « class » permet une mise en forme (avec CSS par exemple)

class="class1 class2 class3"

id="menu"

Autres balises

<div> → Bloc

<span> → Bout de texte inline

<br> → Retour chariot

<nav> → Element de navigation

<blockquote> → Citation

<p> → Paragraphe

Il existe beaucoup d'autres balises, elles ont toute un sens sémantique pour le navigateur et permettent de mieux structurer notre page.

Le CSS

Les feuilles de style en cascade, généralement appelées CSS de l'anglais Cascading Style Sheets, forment un langage informatique qui décrit la présentation des documents HTML. On appelle cela ainsi, car le navigateur va lire le document de haut en bas. Ceux qui ont la plus grande priorité sont en bas. En voici la syntaxe :

Les classes, qu'on a abordées plus haut, vont permettre de styliser précisément un ou plusieurs éléments de la page. Pour cela, on va utiliser les sélecteurs. En CSS désigne une class avec un (.) et un id avec un (#).
